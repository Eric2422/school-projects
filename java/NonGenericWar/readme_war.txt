 *  What is the point of the interface List?  What does it mean for
 *  a Java class (e.g., LinkedList) to implement an interface?
 **********************************************************************/
The List interface provides empty-body methods for LinkedList to implement(e.g. add(), set(), remove())
When a class implements an interface, it uses the interface's method headers
but adds its own code to the method.
